UPSKILLING - SQL SCREENSHOTS

These screenshots were generated from actual SQL query execution against a local
SQLite-compatible mirror of the Community Event Portal dataset. The result windows
are styled as a realistic SQL-client preview; they are not captures from MySQL Workbench.

The dataset includes the handbook sample rows plus extra demo rows so that queries
such as "at least 10 feedback submissions", recent-user queries, duplicate registrations,
and session conflicts produce visible results.

Use the original MySQL .sql files for submission/execution. These PNG files are output previews.
