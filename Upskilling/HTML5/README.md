# Module 1 - HTML5 Hands-on Exercises

## Project Theme
## Local Community Event Portal

This module focuses on creating a browser-based community event portal using HTML5 concepts.

The objective was to understand semantic HTML structure, navigation, forms, media handling, browser storage, geolocation and debugging.

---
